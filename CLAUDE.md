# fake-interview

## コンセプト

会社タイプを選ぶだけで、AIが面接官になって模擬面接をしてくれるアプリ。
個人情報ゼロ、アップロードなし、秒で始まる。

## 技術スタック

- **Frontend**: Next.js 16 (App Router / Turbopack)
- **言語**: TypeScript
- **スタイル**: Tailwind CSS v4
- **認証**: Clerk (Googleログインのみ)
- **AIエージェント**: LangGraph × Claude (Anthropic API)
- **回数制限**: Upstash Redis (1日3回/アカウント)
- **デプロイ**: Vercel

## ディレクトリ構成

```
  app/          # App Router ページ・レイアウト
```

## ユーザーフロー

1. Googleログイン (Clerk)
2. 会社タイプを選ぶ (4択)
3. 規模感を選ぶ (3択)
4. 一次面接 (カルチャーフィット / 3問) → 合否コメント
5. 二次面接 (技術 / 3問) → 合否コメント
6. 最終面接 (意思確認 / 3問) → 総合コメント
7. 結果をXでシェア

## 面接設計

- 質問は1ターン1問のみ
- 面接官の発言は3行以内
- ストリーミングでリアルタイム表示
- 数値スコアなし、コメントのみ
- 面接官キャラは会社タイプとフェーズで変化

## 個人情報方針

- 経歴書アップロードなし
- 面接中の回答はセッション内のみ保持、終了後破棄
- データ保存なし
